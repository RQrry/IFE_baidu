### Web语义化
每个标签做它该做的事。Web语义化是指使用语义恰当的标签,使页面有良好的结构,页面元素有含义,能够让人和搜索引擎都容易理解。

### HTML5新增
HTML5 中的一些有趣的新特性：
* 用于绘画的` canvas` 元素
* 用于媒介回放的 `<video>` 和`< audio>` 元素
* 对本地离线存储的更好的支持
* 新的特殊内容元素，`<header>`，`<footer>`，`<nav>`，`<article>`，`<section>`，`<aside>`
* 新的表单控件，比如 `calendar`、`date`、`time`、`email`、`url`、`search`
